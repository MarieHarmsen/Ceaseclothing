# Ceaseclothing
An iOS app that shows images of clothes to potential clients.


## Additional notes
Used SnapKit for a few UI constraints and SDWebImage for the image downloads. 

## Features
- Image Viewing
- Credit screen
- Loading screen
- Error Screen
